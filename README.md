# Teste para vaga da Locaweb

## Requisitos
#### Ruby 3.0.2
#### Rails 7.1.2
#### Bundler 2.4.22
#### SQLite 3

## Preparação do ambiente
```
bundle install
rails db:create
rails db:migrate
rails db:seed
```

## Execução do ambiente

para execução do servidor
```
rails serve
```

para execução do console
```
rails c
```

para execução dos testes
```
rspec
```
